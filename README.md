TaskMate App
 📌 Overview

This is a simple and user-friendly To-Do List application built using React.
It helps users manage daily tasks by adding, updating, and deleting tasks.

---

 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ✏️ Edit tasks
* ❌ Delete tasks
* 🔍 Filter tasks (Completed / Pending)
* 💾 Save tasks in local storage

---

 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
